# Usage

```shell
m -a xx [dir] # mark dir (default to $(pwd))
m xx # jump to dir
m -l # list all marks
m -d xx # delete mark
```

# Installation

1. install git
2. download the code and source it:
```shell
source /path/to/mdaj/mdaj.sh
```

**Make sure the name 'm' is not occupied by other executable or alias!**
